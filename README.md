# KIOO Airdrop

KIOO's airdrop webapp

## Running Locally

1. Clone the repository

   ```bash
   git clone https://github.com/KIOO-Official/kioo-airdrop.git
   ```
   
2. Start the development server

   ```bash
   go run main.go
   ```
